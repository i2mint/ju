Renders RJSF specifications as Jupyter widgets.

Provides a bridge between RJSF form specifications and ipywidgets, allowing interactive form viewing and data collection in notebooks.

Example:

>>> rjsf_dict = {'rjsf': {'schema': {...}, 'uiSchema': {...}}}
>>> viewer = RJSFViewer(rjsf_dict)  
>>> viewer.display()
